เริ่มใช้งานใน 3 ขั้นตอน

1. สร้างห้องเลือกประเภท: งาน / โปรเจกต์ / เรียนรู้
2. มอบหมายงานกรอกกำหนดส่ง + แนบลิงก์
3. ติดตามผลดูปฏิทินสัปดาห์ ตรวจงาน ให้คะแนน
เริ่มสร้างห้อง เปิด Dashboard

คำถามที่พบบ่อย

เริ่มแบบ Local ได้ทันที (ข้อมูลเก็บในเครื่อง) แล้วค่อยเชื่อมต่อบัญชีภายหลัง

งาน: เช็กลิสต์ + งาน • โปรเจกต์: บอร์ดคานบัง • เรียนรู้: บทเรียน + งาน + คะแนน

ใช่ เมื่อสร้างงานพร้อมวันครบกำหนด จะขึ้นใน “สัปดาห์นี้” ทันที
